A simple Bingo game built using **C# Windows Forms**.

## Description
This is a mini Bingo game where the player generates random numbers and marks them on a 5x5 grid.
The goal is to complete rows, columns, or diagonals before time runs out.

Each completed line reveals one letter from the word **BINGO**.
Completing all five letters wins the game.

## Features
- 5x5 Bingo board with random numbers
- Random number generator
- Time-limited gameplay (60 seconds)
- Visual feedback using colors
- Win detection (rows, columns, diagonals)
- Replay option
- Simple and clean UI

## How to Play
1. Start a new game.
2. Click **Random Number** to generate numbers.
3. Matching numbers on the board will be marked.
4. Complete rows, columns, or diagonals.
5. Finish all five to win before time ends.
